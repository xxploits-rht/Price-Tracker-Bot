 📊 Price Tracker & Alert Bot

A Python-based Price Tracking System that monitors product prices from e-commerce websites and sends Telegram alerts whenever the product price reaches the user's target price.

---

## Features

- Product Price Scraping using Playwright
- Telegram Bot Integration
- MongoDB Database
- Product Tracking
- Target Price Alert
- Price History Storage
- Automatic Price Checking
- Flask Dashboard
- Bootstrap Responsive UI
- Chart.js Price History Graph

---

## Technologies

- Python
- Flask
- MongoDB
- PyMongo
- Telegram Bot API
- Playwright
- Chart.js
- Bootstrap 5

## How to Run

Install dependencies

```
pip install -r requirements.txt
```

Run Flask Dashboard

```
python app.py
```

Run Telegram Bot

```
python bot.py
```

Run Price Checker

```
python price_checker.py
```
